# Account Information
## Logged in account
Your can easily query the information of the currently logged in account by using the following query:

```graphql
query loggedInAccount {
  loggedInAccount {
    id
    name
    slug
  }
}
```

## Any account
Want to query the information for any account?

:::tabs
---Query

```graphql
query account($slug: String) {
  account(slug: $slug) {
    id
    name
    slug
  }
}
```

---Variables

```json
{
  "slug": "znarf"
}
```

:::
